# online-_store
# المتجر الالكتروني:-
مشروع متجر الكتروني.
## 📌 وصف المشروع
نظام متجر الكتروني يسمح للمستخدمين بتصفح المنتجات واضافتها للسلة واتمام الطلب. 
ويوجد لوحة تحكم للادمن لادارة المنتجات والطلبات.

## ✨ المميزات
- **واجهة المستخدم**: عرض المنتجات بشكل كروت + اضافة للسلة
- **سلة المشتريات**: تعديل الكمية وحذف المنتجات وحساب الاجمالي
- **نظام الطلبات**: تسجيل الطلب في قاعدة البيانات
- **لوحة تحكم الادمن**: 
    - اضافة / تعديل / حذف المنتجات
    - رفع صور للمنتجات
    - عرض الطلبات وتحديث حالتها pending/completed/cancelled
- **تسجيل دخول وتسجيل حساب جديد**

## 🛠 التقنيات المستخدمة
- PHP 
- MySQL
- Bootstrap 5
- HTML5, CSS3
